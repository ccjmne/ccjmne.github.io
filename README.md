# ccjmne.github.io

My personal website.  
Not intended as a portfolio nor an online curriculum vitæ, but as a mere playground (abeit organised) for me to quench my thirst for experimentation with the ever-new Vanilla ECMAScript and other platform-native Web front-end goodies. `#UseThePlatform`

... and a bit of TypeScript and Sass.  
Packaged using Webpack.  
Developed for Chromium.

Built with a bit of ❤ and a lot of Functional Programming.
